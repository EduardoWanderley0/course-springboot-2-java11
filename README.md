### Getting started

```bash
# Clone repository
$ git clone https://github.com/eduardowanderleydev/course-springboot-2-java11.git

# go to project directory
$ cd course-springboot-2-java11

# run application
$ mvn spring-boot:run

# The server will start on port:8080
```
